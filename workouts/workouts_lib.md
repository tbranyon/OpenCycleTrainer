This document contains a crude list of MRC workouts that should be generated to populate the default workout library that OCT ships with.  

## Default warmup
The default session warmup sequence should be as follows. All workouts detailed further down should prepend this warmup sequence unless otherwise specified.  
* 2m @ 50%
* 2m @ 60%
* 1m @ 70%
* 2m @ 85%
* 1m @ 92%
* 2m @ 50%

## Default cooldown
Similar to the warmup sequence, all workouts should append this cooldown after intervals unless otherwise specified.
* 5m ramp 55-40%
